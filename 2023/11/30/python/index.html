查找类型 说明
contains 包含某个字/字母的字段（区分大小写），类似于：where name like “%刘%”
icontains 包含，不区分大小写
in id_in=[3,5]，类似于SQL语句：where id in （3,5）
gt 大于 ; id_gt=3 等同于 where id >3
gte 大于等于
lt 小于
lte 小于等于
startswith 匹配字符串开头，区分大小写 name_startswith=“刘”，等同于where name like “刘%”
